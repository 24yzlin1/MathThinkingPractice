# 任务三：Bootstrap 方法 — 估计问题与数据说明

## 1. 估计问题

**目标：** 估计某校全体学生的平均考试成绩（总体均值 μ）。

**背景：** 假设我们无法获取全校所有学生的成绩数据，只能通过随机抽取的 200 名学生作为样本，来推断全校学生的平均考试成绩。由于样本均值 x̄ 会因抽样不同而波动，我们需要一种方法来评估这个估计的可靠性——即给出一个置信区间，告诉我们"总体均值大概落在什么范围内"。

**为什么用 Bootstrap：** 传统的置信区间计算需要假设数据服从正态分布，但实际数据可能不满足这一假设。Bootstrap 方法不需要对总体分布做任何假设，只需从原始样本中有放回地重抽样，就能近似得到统计量的抽样分布，从而计算置信区间。

## 2. 数据来源

**数据集：** Student Exam Scores Dataset（Kaggle）
**链接：** https://www.kaggle.com/datasets/mirzayasirabdullah07/student-exam-scores-dataset
**文件：** data/sample_data.csv

## 3. 样本描述

| 项目   | 说明             |
| ---- | -------------- |
| 样本量  | 200 名学生        |
| 变量数  | 6 个            |
| 数据类型 | 连续型（考试成绩为连续变量） |

## 4. 变量说明

| 变量名                | 含义             | 类型       | 示例值             |
| ------------------ | -------------- | -------- | --------------- |
| student_id         | 学生编号           | 标识变量     | S001, S002, ... |
| hours_studied      | 每周学习时长（小时）     | 连续变量     | 1.3 ~ 12.0      |
| sleep_hours        | 每晚睡眠时长（小时）     | 连续变量     | 4.3 ~ 9.0       |
| attendance_percent | 出勤率（%）         | 连续变量     | 50.5 ~ 97.4     |
| previous_scores    | 之前的成绩          | 连续变量     | 41 ~ 90         |
| **exam_score**     | **考试成绩（目标变量）** | **连续变量** | **18.3 ~ 42.0** |

## 5. 目标统计量

| 统计量             | 说明             |
| --------------- | -------------- |
| 点估计             | 原始样本的考试成绩均值 x̄ |
| 置信区间            | 95% 百分位数置信区间   |
| Bootstrap 重采样次数 | B = 1000（可配置）  |

## 6. 估计假设

1. **独立同分布（i.i.d.）：** 假设 200 个学生的成绩是独立抽取的，每个学生来自同一总体分布。
2. **样本代表性：** 假设这 200 名学生是全校学生的随机样本，能够代表总体。
3. **有放回抽样：** Bootstrap 重采样时，每次从 200 个样本中有放回地抽取 200 个，允许重复。

## 7. Bootstrap 方法步骤

1. 从原始样本（200 个 exam_score）中有放回地随机抽取 200 个值，构成一个 Bootstrap 样本。
2. 计算该 Bootstrap 样本的均值。
3. 重复步骤 1-2 共 B = 1000 次，得到 1000 个 Bootstrap 均值。
4. 将这 1000 个均值从小到大排序，取第 2.5% 和 97.5% 分位数，得到 95% 置信区间。
5. 点估计为原始样本均值 x̄。

## 8. 预期输出

- 点估计值（原始样本均值）
- 95% 置信区间（下限, 上限）
- Bootstrap 均值分布直方图
- 置信区间示意图
