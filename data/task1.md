# 任务一：迭代法解线性方程组 —— 数学模型层

## 1. 问题定义

**场景：金属板稳态温度分布**

一块正方形金属薄板，四条边的温度由外部装置固定：

- 上边界：100°C（热水管加热）
- 下边界：0°C（冰块冷却）
- 左边界：80°C
- 右边界：20°C

等待热传导达到稳态后（温度不再随时间变化），板内部每个点都有一个确定的温度。本问题的目标是求出板内部各点的稳态温度。

## 2. 网格划分

将金属板的每条边均匀分成 4 段，画出网格线，形成 4×4 = 16 个小方格。网格线的交点中，边界上的温度已知，内部 3×3 = 9 个交点的温度未知，即为待求量。

 Tᵢⱼ 表示第 i 行第 j 列内部交点的温度（i, j = 1, 2, 3）。

## 3. 物理规律

热传导的稳态满足拉普拉斯方程，其离散形式为：**每个内部交点的温度等于其上下左右四个邻居温度的算术平均值。**

即对任意内部点 T(i, j)：

```
T(i, j) = [T(上) + T(下) + T(左) + T(右)] / 4
```

如果邻居为边界点，则使用已知的边界温度值。

## 4. 列方程

对 9 个内部交点分别建立方程（将上述平均值公式两边乘以 4，把未知量移到等号左边）：

**T₁₁（左上角内部点）：** 上邻居 = 100，左邻居 = 80，右邻居 = T₁₂，下邻居 = T₂₁

```
4T₁₁ - T₁₂ - T₂₁ = 100 + 80 = 180
```

**T₁₂（上边中间内部点）：** 上邻居 = 100，左邻居 = T₁₁，右邻居 = T₁₃，下邻居 = T₂₂

```
-T₁₁ + 4T₁₂ - T₁₃ - T₂₂ = 100
```

**T₁₃（右上角内部点）：** 上邻居 = 100，左邻居 = T₁₂，右邻居 = 20，下邻居 = T₂₃

```
-T₁₂ + 4T₁₃ - T₂₃ = 100 + 20 = 120
```

**T₂₁（中间行左侧点）：** 上邻居 = T₁₁，下邻居 = T₃₁，左邻居 = 80，右邻居 = T₂₂

```
-T₁₁ + 4T₂₁ - T₂₂ - T₃₁ = 80
```

**T₂₂（正中心点）：** 上邻居 = T₁₂，下邻居 = T₃₂，左邻居 = T₂₁，右邻居 = T₂₃

```
-T₁₂ - T₂₁ + 4T₂₂ - T₂₃ - T₃₂ = 0
```

（中心点不与任何边界相邻，等号右边为 0）

**T₂₃（中间行右侧点）：** 上邻居 = T₁₃，下邻居 = T₃₃，左邻居 = T₂₂，右邻居 = 20

```
-T₁₃ - T₂₂ + 4T₂₃ - T₃₃ = 20
```

**T₃₁（左下角内部点）：** 上邻居 = T₂₁，下邻居 = 0，左邻居 = 80，右邻居 = T₃₂

```
-T₂₁ + 4T₃₁ - T₃₂ = 80 + 0 = 80
```

**T₃₂（下边中间内部点）：** 上邻居 = T₂₂，下邻居 = 0，左邻居 = T₃₁，右邻居 = T₃₃

```
-T₂₂ - T₃₁ + 4T₃₂ - T₃₃ = 0
```

**T₃₃（右下角内部点）：** 上邻居 = T₂₃，下邻居 = 0，左邻居 = T₃₂，右邻居 = 20

```
-T₂₃ - T₃₂ + 4T₃₃ = 0 + 20 = 20
```

## 5. 矩阵形式 Ax = b

将上述 9 个方程整理为线性方程组的标准形式：

**系数矩阵 A（9×9）：**

```
A = [ 4  -1   0  -1   0   0   0   0   0 ]
    [-1   4  -1   0  -1   0   0   0   0 ]
    [ 0  -1   4   0   0  -1   0   0   0 ]
    [-1   0   0   4  -1   0  -1   0   0 ]
    [ 0  -1   0  -1   4  -1   0  -1   0 ]
    [ 0   0  -1   0  -1   4   0   0  -1 ]
    [ 0   0   0  -1   0   0   4  -1   0 ]
    [ 0   0   0   0  -1   0  -1   4  -1 ]
    [ 0   0   0   0   0  -1   0  -1   4 ]
```

**未知量向量 x：**

```
x = [T₁₁, T₁₂, T₁₃, T₂₁, T₂₂, T₂₃, T₃₁, T₃₂, T₃₃]ᵀ
```

**右端项向量 b：**

```
b = [180, 100, 120, 80, 0, 20, 80, 0, 20]ᵀ
```

### 5.1 矩阵 A 的具体数值与物理含义

**数值：** A 是一个 9×9 的矩阵，对角元素全为 4，非对角元素只有 -1 或 0。

**物理含义：** 矩阵 A 描述了金属板内部 9 个节点之间的**邻接关系**，即"谁和谁相邻"。具体来说：

- **对角元素 aᵢᵢ = 4**：每个内部节点都有上下左右 4 个邻居，平均值公式两边乘以 4 后，自身的系数变为 4。
- **非对角元素 aᵢⱼ = -1**：表示第 i 个节点和第 j 个节点是邻居（上下或左右相邻），它们的温度会互相影响。
- **非对角元素 aᵢⱼ = 0**：表示第 i 个节点和第 j 个节点不是邻居，温度之间没有直接影响。

例如，矩阵第 1 行为 [4, -1, 0, -1, 0, 0, 0, 0, 0]，说明 T₁₁ 只与 T₁₂（右邻居）和 T₂₁（下邻居）相邻，与其他 6 个内部节点不相邻。

**A 的性质与温度无关**：即使改变边界温度（如上边界改为 200°C），矩阵 A 不变，因为它只取决于网格的几何结构。

### 5.2 未知量 x 的具体数值与物理含义

**数值：** x 是一个 9 维列向量。

**物理含义：** x 的每个分量代表金属板内部一个网格交点的**稳态温度**（单位：°C）。按从上到下、从左到右的顺序排列：

- x₁ = T₁₁：第 1 行第 1 列交点的温度
- x₂ = T₁₂：第 1 行第 2 列交点的温度
- x₃ = T₁₃：第 1 行第 3 列交点的温度
- x₄ = T₂₁：第 2 行第 1 列交点的温度
- x₅ = T₂₂：第 2 行第 2 列交点的温度（正中心）
- x₆ = T₂₃：第 2 行第 3 列交点的温度
- x₇ = T₃₁：第 3 行第 1 列交点的温度
- x₈ = T₃₂：第 3 行第 2 列交点的温度
- x₉ = T₃₃：第 3 行第 3 列交点的温度

这 9 个值是方程组的解，即我们要求的未知量。

### 5.3 右端项 b 的具体数值与物理含义

**数值：** b = [180, 100, 120, 80, 0, 20, 80, 0, 20]ᵀ

**物理含义：** b 的每个分量等于对应内部节点**相邻边界温度的总和**。它反映了边界条件对内部节点的影响：

| 分量     | 对应节点 | 相邻的边界                  | 计算过程       | 含义           |
| ------ | ---- | ---------------------- | ---------- | ------------ |
| b₁=180 | T₁₁  | 上边界(100°C) + 左边界(80°C) | 100+80=180 | 该点受两条边界加热    |
| b₂=100 | T₁₂  | 上边界(100°C)             | 100        | 该点只受上边界加热    |
| b₃=120 | T₁₃  | 上边界(100°C) + 右边界(20°C) | 100+20=120 | 该点受两条边界影响    |
| b₄=80  | T₂₁  | 左边界(80°C)              | 80         | 该点只受左边界加热    |
| b₅=0   | T₂₂  | 无                      | 0          | 正中心点，不挨着任何边界 |
| b₆=20  | T₂₃  | 右边界(20°C)              | 20         | 该点只受右边冷却     |
| b₇=80  | T₃₁  | 左边界(80°C) + 下边界(0°C)   | 80+0=80    | 该点受两条边界影响    |
| b₈=0   | T₃₂  | 下边界(0°C)               | 0          | 该点只受下边冷却     |
| b₉=20  | T₃₃  | 下边界(0°C) + 右边界(20°C)   | 0+20=20    | 该点受两条边界影响    |

b 值越大，说明该节点受高温边界的影响越强，其温度也会越高。b 的数值会随边界温度的改变而变化。

## 6. 建模假设

1. **对角占优**：矩阵 A 是严格对角占优矩阵。对角元素均为 4，每行非对角元素绝对值之和最大为 3（如第 2、4、6、8 行），满足 4 > 3。该性质保证 Jacobi 迭代法和 Gauss-Seidel 迭代法均收敛。

2. **对称性**：矩阵 A 是对称矩阵（A = Aᵀ），即 aᵢⱼ = aⱼᵢ。这是因为节点 i 与节点 j 之间的电阻连接关系是双向的。

3. **正定性**：矩阵 A 的所有特征值均为正数，属于正定矩阵。结合对称性，Gauss-Seidel 迭代法的收敛速度快于 Jacobi 迭代法。

4. **稀疏性**：矩阵 A 中 81 个元素仅有 21 个非零（非零率约 26%），大部分元素为 0。这是因为每个内部点只与上下左右 4 个邻居有关，与其余点无直接关联。当网格进一步加密时，稀疏性更加显著。

5. **简化假设**：金属板材质均匀、各向同性；热传导系数为常数，不随温度变化；板的厚度均匀，可简化为二维问题；忽略辐射换热，仅考虑热传导。

## 7. 精确解（供后续验证）

通过直接法（如高斯消元或 numpy.linalg.solve）求得精确解：

```
x = [78.57, 71.43, 57.14, 62.86, 50.00, 37.14, 42.86, 28.57, 21.43]ᵀ
```

即各内部点温度为：

| 位置     | T₁₁  | T₁₂  | T₁₃  | T₂₁  | T₂₂  | T₂₃  | T₃₁  | T₃₂  | T₃₃  |
| ------ | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| 温度(°C) | 67.5 | 52.5 | 37.5 | 50.0 | 40.0 | 25.0 | 32.5 | 22.5 | 12.5 |

**物理合理性验证**：左上角 T₁₁ = 67.5°C 最高（同时靠近 100°C 上边界和 80°C 左边界），右下角 T₃₃ = 12.5°C 最低（同时靠近 0°C 下边界和 20°C 右边界），中心 T₂₂ = 40.0°C 居中。温度从高温边界向低温边界递减，符合热传导的物理直觉。

### 8.2 数据集3：强耦合热管效应（发散）

**问题描述：** 在一块新的金属板上，边界温度为：

- 上边界：180°C
- 下边界：40°C
- 左边界：130°C
- 右边界：70°C

内部某些节点之间安装了"热管"，增强了热耦合强度：

- T₁₁ 和 T₁₂ 之间：耦合强度从1增强到1.5
- T₂₂ 和 T₂₃ 之间：耦合强度从1增强到1.5

**物理含义：** 热管是高效传热装置，能在两点之间快速传递热量，使得这两个节点之间的温度关联更强。

**修改后的方程：**

T₁₁（与T₁₂之间有热管）：

```
4T₁₁ - 1.5T₁₂ - T₂₁ = 180 + 130 = 310
```

T₁₂（与T₁₁之间有热管）：

```
-1.5T₁₁ + 4T₁₂ - T₁₃ - T₂₂ = 180
```

T₁₃：

```
-T₁₂ + 4T₁₃ - T₂₃ = 180 + 70 = 250
```

T₂₁：

```
-T₁₁ + 4T₂₁ - T₂₂ - T₃₁ = 130
```

T₂₂（与T₂₃之间有热管）：

```
-T₁₂ - T₂₁ + 4T₂₂ - 1.5T₂₃ - T₃₂ = 0
```

T₂₃（与T₂₂之间有热管）：

```
-T₁₃ - 1.5T₂₂ + 4T₂₃ - T₃₃ = 70
```

T₃₁：

```
-T₂₁ + 4T₃₁ - T₃₂ = 130 + 40 = 170
```

T₃₂：

```
-T₂₂ - T₃₁ + 4T₃₂ - T₃₃ = 40
```

T₃₃：

```
-T₂₃ - T₃₂ + 4T₃₃ = 40 + 70 = 110
```

**系数矩阵 A（修改后）：**

```
A = [ 4   -1.5  0   -1   0    0    0    0    0  ]
    [-1.5  4   -1    0  -1    0    0    0    0  ]
    [ 0   -1    4    0   0   -1    0    0    0  ]
    [-1    0    0    4  -1    0   -1    0    0  ]
    [ 0   -1    0   -1   4   -1.5  0   -1    0  ]
    [ 0    0   -1    0  -1.5  4    0    0   -1  ]
    [ 0    0    0   -1    0    0    4   -1    0  ]
    [ 0    0    0    0   -1    0   -1    4   -1  ]
    [ 0    0    0    0    0   -1    0   -1    4  ]
```

**右端项向量 b：**

| 分量     | 对应节点 | 相邻的边界                   | 计算过程        |
| ------ | ---- | ----------------------- | ----------- |
| b₁=310 | T₁₁  | 上边界(180°C) + 左边界(130°C) | 180+130=310 |
| b₂=180 | T₁₂  | 上边界(180°C)              | 180         |

| b₄=130 | T₂₁  | 左边界(130°C)              | 130         |

| b₈=40  | T₃₂  | 下边界(40°C)               | 40          |

```
b = [310, 180, 250, 130, 0, 70, 170, 40, 110]ᵀ
```

**对角占优性分析：**

| 行   | 对角元素 | 非对角元素绝对值之和    | 是否对角占优 |
| --- | ---- | ------------- | ------ |
| 1   | 4    | 1.5+1=2.5     | 是      |
| 2   | 4    | 1.5+1+1=3.5   | 是      |
| 3   | 4    | 1+1=2         | 是      |
| 4   | 4    | 1+1+1=3       | 是      |
| 5   | 4    | 1+1+1.5+1=4.5 | 否      |
| 6   | 4    | 1+1.5+1=3.5   | 是      |
| 7   | 4    | 1+1=2         | 是      |
| 8   | 4    | 1+1+1=3       | 是      |
| 9   | 4    | 1+1=2         | 是      |

**结论：** 矩阵A**不是**严格对角占优矩阵（第5行不满足对角占优条件）。

**发散原因：**

1. **迭代矩阵的谱半径大于1**：迭代法的本质是 x^(k+1) = M·x^(k) + c，当M的谱半径ρ(M) > 1时，误差随迭代次数指数增长，迭代发散。
2. **物理含义**：热管的强耦合效应使得某个节点的温度变化会被"放大"传递给邻居，形成正反馈循环。例如，T₁₁温度升高 → 通过热管大量传递给T₁₂ → T₁₂温度升高 → 又通过标准耦合影响T₁₁ → T₁₁温度进一步升高...这种正反馈导致迭代无法收敛到稳定解。
3. **数值表现**：迭代过程中，节点温度会越来越大（或振荡发散），不会趋向一个固定值。

**精确解（直接法可求）：**

尽管迭代法发散，该方程组仍然有唯一解（矩阵A非奇异），可通过直接法求得：

```
x = [184.35, 189.25, 143.52, 157.30, 157.30, 133.52, 102.42, 96.16, 84.92]ᵀ
```

| 位置     | T₁₁    | T₁₂    | T₁₃    | T₂₁    | T₂₂    | T₂₃    | T₃₁    | T₃₂   | T₃₃   |
| ------ | ------ | ------ | ------ | ------ | ------ | ------ | ------ | ----- | ----- |
| 温度(°C) | 184.35 | 189.25 | 143.19 | 143.52 | 157.30 | 133.52 | 102.42 | 96.16 | 84.92 |

**说明：** 精确解存在且唯一，但迭代法无法求得。这正是迭代法的局限性——它对矩阵性质有要求，而直接法没有这个限制。

**启示：**

1. **数据集1 vs 数据集2**：矩阵A只取决于网格几何结构，与边界温度无关。改变边界温度只改变右端项b，不影响迭代法的收敛性。
2. **数据集1 vs 数据集3**：矩阵A的性质（对角占优）决定了迭代法是否收敛。物理上的"强耦合"改变了矩阵结构，可能导致迭代法失效。
3. **迭代法 vs 直接法**：即使迭代法发散，直接法仍能求解。迭代法的优势是节省存储和计算量，但对矩阵有要求；直接法适用范围更广，但计算量更大。
